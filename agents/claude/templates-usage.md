# Templates 使用说明 — Claude

`agents/templates/` 下的模板由"初始化项目"过程自动使用。

如果你要手动使用，每个模板对应一类文件：

| 模板 | 用途 |
|------|------|
| `project.ai.md` | 项目规则文件模板 |
| `project-meta.yaml` | 项目元数据，`project_init` 的 meta 参数 |
| `tone.md` | 文风设定和角色声线模板 |
| `prompts.md` | 推演提示词模板（四个场景） |
| `review.md` | 审查模板（反AI/角色/声线/剧情/节奏/对话） |
| `location.md` | 地点档案模板（初始外观→变更记录→当前状态；用 MCP 工具管理数据，文件做备份） |
| `db-schema.md` | 数据库表结构参考文档 |
| `chapter-outline.md` | 大纲格式 |
| `chapter-draft.md` | 正文格式 |
| `character-profile.md` | 人物画像格式 |
