# 推演提示词模板

> 每次写新章节前，用这些模板构建推演上下文，辅助剧情推演。
> 根据项目类型（原创/同人）选择适用的模板。

---

## 模板一：章节推演（核心模板）

```markdown
# 角色状态

## {主角}（当前状态）
{从人物追踪文件读取}

## {重要角色}（当前状态）
{从人物追踪文件读取}

## 其他在场角色
{根据场景需要读取}

# 当前剧情位置
上一章结尾：{摘要}
当前地点：{地点}
当前时间：{时间点}

# 即将到来的关键事件
事件：{描述}
前置条件：{条件列表}

# 推演任务

请根据以上角色状态和事件信息，推演以下内容：

1. **{主角}的行动判断**：根据角色性格和当前动机，他会怎么做？
2. **{重要角色}的反应**：根据性格和情绪状态，对主角的行动会如何反应？
3. **剧情走向推演**：这个事件可能如何发展？
4. **角色状态变更**：列出所有角色的状态变化：
   - 关系变化
   - 新增信息/认知
   - 情绪变化
   - 动机变化

# 约束
- 角色行为必须符合其性格画像
- 角色只能基于自己"知道的信息"做决策（禁止全知视角）
- 主角不能突然变强或获得不合理的能力
```

## 模板二：角色心理推演（细化模板）

```markdown
# 场景设定
{描述当前场景}

# 角色当前心理状态
- 情绪基调：{从人物追踪读取}
- 对{相关角色}的关系值：{如信任度0-10}
- 当前核心冲突：{角色内心的矛盾点}

# 发生了什么
{描述触发事件}

# 推演要求

请从三个视角分别推演该角色的反应：

## 视角1：外在反应
他会说什么？做什么？表情和身体语言是什么？
（注意：角色的外在反应可能与内心相反。）

## 视角2：内心独白
他心里在想什么？他自己意识到了什么？
（注意：角色可能会压抑真实情感。）

## 视角3：第三方观察者
如果有旁观者（或读者），会观察到什么？
旁观者对这一幕的理解是什么？

## 状态变更建议
- 关系变化：+/- 多少？
- 角色是否有自我认知的突破？
- 后续行为会有什么无意识的改变？
```

## 模板三：剧情分支推演（适用于同人正典或原创分支）

```markdown
# 正典/主线事件
ID：{ID}
名称：{名称}
原有走向：{完整描述}
前置条件：{条件列表}

# 主角的干预
主角做了什么/打算做什么：{描述}
干预的动机：{为什么要干预}

# 当前世界状态
- 各角色位置：{位置列表}
- 各角色当前关系：{关系状态}
- 已经偏移的其他事件：{之前已经改变的事件列表}

# 推演要求

1. **因果链分析**：干预会导致哪些因果链变化？
   - 直接影响：{这个事件本身怎么变}
   - 连锁影响：{后续哪些事件会受影响}
   - 不可逆影响：{哪些变化一旦发生就无法挽回}

2. **各角色反应**：每个涉及角色会如何应对变化？
   - 按角色性格画像推断
   - 考虑角色当前的知识范围

3. **风险评估**：这个干预有什么风险？
   - 是否可能导致更糟的结果？
   - 主角掌握的信息是否足够支撑这个决策？
   - 有没有没考虑到的变量？

4. **替代事件**：如果原事件被取消/改变，什么事件会填补这个叙事空间？
```

## 模板四：章节生成（最终输出）

```markdown
# 章节信息
章节号：第{N}章
标题：{标题}
视角角色：{POV}
时间线位置：{在故事时间线中的位置}
前一章摘要：{上一章发生了什么}

# 本章要实现的目标
- 剧情推进：{要发生什么事件}
- 角色发展：{哪个角色要有什么变化}
- 关系变化：{哪对关系要推进}
- 情绪曲线：{本章的情绪走向}

# 角色状态加载
{从人物追踪读取所有在场角色的当前状态}

# 场景列表
1. {场景1描述}
2. {场景2描述}
3. {场景3描述}

# 文风要求
- 参见 style/tone.md
- 本章重点氛围：{沉重/温暖/紧张/日常...}

# 写作约束
- 字数目标：{目标字数}
- 主角的对白要符合角色设定
- 所有角色只能基于自己知道的信息行动
- {其他项目特定约束}

# 生成后需要更新的数据
- 人物状态（character_snapshot）
- 伏笔登记（如有新伏笔）
- 时间线（如有事件推进）
```
